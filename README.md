# Strava.cz Food Previews

A browser extension for Chromium-based browsers and Mozilla Firefox that fetches and inserts live food images directly into the `app.strava.cz` lunch ordering panel. It queries DuckDuckGo's layout engine to give you a visual preview of your meal options.
